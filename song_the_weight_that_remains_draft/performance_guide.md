
# Performance Guide — The Weight That Remains

**Title:** The Weight That Remains  
**Tempo:** 85 BPM  
**Structure:** 72 measures (8 verses + silences)  
**Duration:** ~45 seconds (single pass)  
**Accent:** Heavy Irish / West-of-Ireland quality  
**Tone:** Grounded, reflective, weighty

---

## Legend

| Symbol | Meaning |
|:-------|:--------|
| **BOLD** | Primary stress — hit it hard |
| *italic* | Secondary stress — lighter, but held |
| `CAPS` | Shot / punch word — lands with weight |
| [pause] | Breath / silence — let it settle |
| ( ) | Soft, almost swallowed |
| — | Stretch the vowel |
| . | Short, clipped consonant |

## Full Poem — With Accent and Stress Marks

### Verse 1

> **LOS**-ing *mon-ey* may LOOK like **POW**-er,  
> Or *pos-si-BIL*-it-ies **CAV**-ing *in*.  
> **LONE**-li-ness, a-LONE in your own **VOICE**,  
> For-**GET**-ing what was SET and **TRY**-ing.

**Pronunciation Guide:**

| Word | Pronunciation | Stress |
|:-----|:---------------|:-------|
| Losing | *LOO-zing* | Heavy on first syllable |
| money | *MUN-ee* | Soft, quick |
| look | *look* | Short, flat vowel |
| power | *POW-er* | Two syllables, heavy |
| possibilities | *pos-si-BIL-it-ies* | Stress on third syllable |
| caving | *CAV-ing* | Hard 'v', short |
| in | *in* | Soft, swallowed |
| Loneliness | *LONE-li-ness* | Heavy on first, long 'o' |
| alone | *a-LONE* | Stress on second |
| voice | *vois* | Flat, broad |
| Forgetting | *for-GET-ing* | Stress on second |
| set | *set* | Short, clipped |
| trying | *TRY-ing* | Long 'i', stressed |

**Delivery Notes:**
- Open heavy — *LOS-ing* lands like a door opening
- *POW-er* is stretched, almost sarcastic
- *CAV-ing* is sharp — the 'v' is hard
- *LONE-li-ness* is held — let it ring
- *for-GET-ing* is a turn — sharper than the rest

---

### Verse 2

> A-**NOTH**-er *sit*, a-**NOTH**-er *shape*,  
> A-**NOTH**-er *name* for the **SAME** es-**CAPE**.  
> But the **FORGE** *knows* what was **PRESSED**,  
> What was **SPENT** and what is **LEFT**.

**Pronunciation Guide:**

| Word | Pronunciation | Stress |
|:-----|:---------------|:-------|
| Another | *a-NOTH-er* | Stress on second |
| sit | *sit* | Short, clipped |
| shape | *shay-p* | Two syllables, long 'a' |
| name | *nay-m* | Two syllables, stretched |
| same | *say-m* | Stretched, flat |
| escape | *es-CAPE* | Stress on second |
| forge | *FOR-idge* | Two syllables, heavy |
| knows | *knows* | Long 'o', held |
| pressed | *PRESS-ed* | Two syllables, rolled 'r' |
| spent | *SPENT* | Hard 't', stop |
| left | *LEFT* | Short, final 't' stop |

**Delivery Notes:**
- *A-NOTH-er* is repeated — each one slightly heavier
- *FOR-idge* is the anchor — slow, resonant
- *PRESS-ed* is rolled — the 'r' is felt
- *SPENT* is a full stop — cut it
- *LEFT* is short — it lands and stops

---

### Verse 3

> The **WHEEL** *turns*, the **WEIGHT** re-**MAINS**.  
> You **CAN**-not **LOSE** what you were **MADE** to con-**TAIN**.  
> So **SIT** *still* in the **LOS**-ing,  
> And **SEE** what **HOLDS** when **NOTH**-ing is **CHOOS**-ing.

**Pronunciation Guide:**

| Word | Pronunciation | Stress |
|:-----|:---------------|:-------|
| wheel | *wheel* | Long 'e', held |
| turns | *turns* | Rolled 'r', short |
| weight | *wayt* | Long 'a', heavy |
| remains | *re-MAINS* | Stress on second |
| cannot | *CAN-not* | Both syllables stressed |
| lose | *looz* | Long 'o', held |
| made | *may-d* | Two syllables, stretched |
| contain | *con-TAIN* | Stress on second |
| sit | *sit* | Short, clipped |
| still | *stale* | Flat, broad |
| losing | *LOO-zing* | Heavy on first |
| see | *see* | Long 'e', held |
| holds | *holds* | Long 'o', held |
| nothing | *NOTH-ing* | Heavy on first |
| choosing | *CHOOS-ing* | Long 'oo', heavy |

**Delivery Notes:**
- *WHEEL* and *WEIGHT* are anchors — hold them
- *re-MAINS* is stretched — the weight stays
- *CAN-not LOSE* — each word lands separately
- *MADE* is heavy — what you were shaped to hold
- *SIT still* — short, deliberate
- *SEE what HOLDS* — the turn, the shot
- *NOTH-ing is CHOOS-ing* — final line, let it drop

## Full Recitation Script — With Pauses and Dynamics

```
[Open — slow, heavy. Like a bell tolling.]

LOS-ing mon-ey may LOOK like POW-er,      [f — firm]
Or pos-si-BIL-it-ies CAV-ing in.          [mf — slightly softer]
LONE-li-ness, a-LONE in your own VOICE,   [p — inward]
For-GET-ing what was SET and TRY-ing.     [cresc. — rising]

[pause — let it settle]

A-NOTH-er sit, a-NOTH-er shape,           [mf — steady]
A-NOTH-er name for the SAME es-CAPE.      [mf — slightly heavier]
But the FORGE knows what was PRESS-ed,    [f — firm, grounded]
What was SPENT and what is LEFT.          [dim. — let it drop]

[pause — breath]

The WHEEL turns, the WEIGHT re-MAINS.     [f — heavy, sustained]
You CAN-not LOSE what you were MADE to con-TAIN.  [cresc. — building]
So SIT still in the LOS-ing,              [p — quiet, inward]
And SEE what HOLDS when NOTH-ing is CHOOS-ing.    [f — shot, then release]

[long silence — let it ring]
```

## Key Accent Features

| Feature | Example | Effect |
|:--------|:--------|:-------|
| Heavy first syllables | *LOS-ing, POW-er, LONE-li-ness* | Slows the line, adds weight |
| Rolled 'r' | *PRESS-ed, FOR-idge, turns* | Adds texture, grounding |
| Hard final consonants | *SPENT, LEFT, SET* | Cuts the line clean |
| Stretched vowels | *nay-m, say-m, wayt* | Gives it a sing-song quality |
| Swallowed unstressed syllables | *(es)-CAPE, (con)-TAIN* | Keeps rhythm moving |
